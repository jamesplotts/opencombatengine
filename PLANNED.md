# Planned Features

This document outlines the future direction of the OpenCombatEngine.

## Phase 3: Content System
- [ ] **5e.tools Import**: Support for importing content from 5e.tools JSON format.
- [ ] **Open5e API**: Integration with Open5e API for SRD content.
- [ ] **Foundry VTT**: Compatibility with Foundry VTT data structures.
- [ ] **Native Format**: Definition of a `.ocf` (Open Combat Format) for portable content.

## Phase 4: Advanced Features
- [ ] **Spell System**: Spell slots, casting levels, components (V, S, M), and school logic.
- [ ] **Class Features**: Rage, Sneak Attack, Wild Shape, etc.
- [ ] **Magic Items**: Attunement, charges, and passive bonuses.
- [ ] **Environmental Effects**: Difficult terrain, cover, visibility/lighting.

## Phase 5: Grid & Tactical
- [ ] **Grid System**: 5ft square grid logic.
- [ ] **Pathfinding**: A* or Dijkstra for movement validation.
- [ ] **Line of Sight**: Raycasting for visibility checks.
